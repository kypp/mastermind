Mastermind
==========

A classical Mastermind game in C using OpenGL (freeglut).

### Controls

Keys: | Action:
--- | ---
**1**-**6** | _Choose color_
**Left**/**Right** | _Move marker_
**Return** | _Approve line_
**q**/**Esc** | _Quit_
